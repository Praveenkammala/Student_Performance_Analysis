# Student Performance Data Analysis Project

## Overview
This project analyzes student academic performance data using Python, SQL, and Microsoft Excel.
It demonstrates an end-to-end data analytics workflow including data cleaning, analysis,
visualization, and automated reporting.

## Tools & Technologies
Python, SQL (MySQL), Microsoft Excel, Pandas, Matplotlib, OpenPyXL

## Project Structure
Student-Performance-Analysis/
- data/raw        : Original datasets
- data/cleaned    : Cleaned datasets
- python/         : Python scripts
- sql/            : SQL scripts
- excel/          : Generated Excel report
- graphs/         : Output visualizations

## Workflow
1. Data cleaning and validation using Python
2. Data analysis and visualization using Python
3. SQL-based analysis using joins and aggregations
4. Automated Excel dashboard generation

## Outputs
- Cleaned datasets in data/cleaned
- Excel dashboard with charts
- Visualizations stored in graphs/

## How to Run the Project
python python/data_cleaning.py
python python/analysis_visualization.py
python python/excel_report_generator.py

## Key Insights
- Attendance strongly impacts academic performance
- Subject-wise and department-wise variations observed
- Pass vs Fail distribution highlights performance gaps
